🚀 Deploying Kubepwn Lab

Kubepwn is a fully-featured Kubernetes Red Team simulation environment. Follow the steps below to launch your lab.

📁 Directory Structure

Kubepwn ├── app.py ├── deploy.py ├── Dockerfile ├── kind-config.yaml ├── kubepwn-app.yaml ├── README.md ├── static/ └── templates/

⚙️ Lab Deployment

  1. Ensure Kind, kubectl, Docker & Python 3 are installed on your system.
  2. Run the deploy script:
python3 deploy.py
💡 Note: The script sets up your Kubernetes cluster using kind, applies manifests, and exposes the vulnerable web app on http://localhost:8080 or NodePort 30080.

🧠 What's Inside?

  • Multiple vulnerabilities including RCE, SSRF, SSTI, and File Upload
  • Post-exploitation stages like privilege escalation, persistence, data exfiltration
  • Cluster-wide compromise simulation
  • Full MITRE ATT&CK mapping
